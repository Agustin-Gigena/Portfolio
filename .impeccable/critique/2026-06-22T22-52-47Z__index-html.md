## Anti-Patterns Verdict

The site no longer has the three loudest AI tells: gradient text is gone, side-stripe borders are gone, emoji icons are SVGs. The stats section (hero-metric template) has been removed entirely. The ghost-card hover has been fixed (no more 1px border + 24px shadow). JetBrains Mono dead weight removed. The design system now matches the implementation.

- **Where:** Global
- **Why:** Power users navigate with keyboard. No `Esc` closes hamburger. No skip link for screen readers.
- **Fix:** Add `Esc` handler for hamburger menu. Add skip-to-content link as first focusable element.
- **Suggested:** `$impeccable audit`

### P2 — Blog placeholder remains
- **Where:** `#blog` section
- **Why:** "Próximamente" signals incomplete site. Visitors make quick trust decisions.

- **Where:** About section
- **Why:** IT recruiters may not know these terms. Creates unnecessary cognitive load.
